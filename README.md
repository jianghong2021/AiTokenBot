#### 决策层（Strategy Engine）

- 策略 1：
动态价格区间（最适合套现）
	- 逻辑： 预设一个绝对价格或基于均线（EMA/MA）的比例。
	- 示例： 本地监控 SOL/USDC 价格。当价格 $\ge \$180$ 且手里持有 SOL 时
	- 触发卖出套现；当价格 $\le \$150$ 且手里有 USDC 时，触发抄底买入。

- 策略 2：
网格交易（Grid Trading，赚取震荡收益）
	- 逻辑： 将主流币在一定价格区间内划分成网格，分批挂单或分批市价执行。
	- 示例： 只要 SOL 在 $160 - $170 之间震荡，每跌 1% 就用 USDC 补仓一次，每涨 1% 就套现卖出一次。
- 策略 3：
自适应移动止盈（Trailing Take Profit）
	- 逻辑： 防止卖早了。当价格达到你的套现线后，机器人不立刻卖出，而是开启“追踪”。
	- 示例： 设定止盈点为 $180，回撤 1% 卖出。如果价格冲到 $185 后回调到 $183.15（回撤了 1%），机器人在 $183.15 本地签名卖出，帮你多赚取利润。
